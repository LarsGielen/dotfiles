# Dotfiles

## Installed programs
- Hyprland
- kitty
- rofi-lbonn-wayland-git
- timeshift
- gnu stow
- nano

# Usage
Clone this repo and use the stow command on the files you want to use

```
stow hyprland
stow terminal
```
